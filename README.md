### MLB Prop Betting Model
Multi-source Python pipeline for MLB player prop analysis. Pulls from 4 live APIs to identify edge against closing lines.

**Data sources:**
- MLB Stats API — player stats & matchups
- Baseball Savant — Statcast metrics
- OpenWeatherMap — real-time ballpark conditions
- The Odds API — live lines & closing line value
